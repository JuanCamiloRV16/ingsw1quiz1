Representación

Es una orientacion al objeto de la página web. [1]

Estructura

Facilita una representación estructurada del documento. [1]

Enlace

Esencialmente, conecta las páginas web a scripts o lenguajes de programación. [1]

Descripcion

Sirve como interfaz de programación para los documentos HTML y XML. da una representación del documento como un grupo de nodos y objetos estructurados que tienen propiedades y métodos. [1]

Imagen tomada de: https://www.todo-argentina.net/cursos/dhtml/pagina2.php

Como se accede/modifica

Mediante lenguajes de script como JavaScript u otros como Python.

Conclusiones

1.

Al otorgar una forma ordenada y estandarizada de modificar documentos HTML, considero que facilita mucho la programación de páginas web.

2.

Como jerarquiza la informacion del documento, ayuda a ver de forma mucho mas intuitiva que etiqueta es la que se debe modificar para dar un comportamiento o diseño especifico.

3.

Permite el uso de las paginas web y los navegadores como medio para mostrar las interfaces graficas del software desarrollado, ya que el DOM permite programar y mostrar estos elementos.
